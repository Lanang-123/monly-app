# Transaction Management App (Monly AI)

A modern, full-featured transaction management application built with Next.js, TypeScript, and Zustand. Features intelligent caching, multi-tab synchronization, and real-time updates.


## Features

### Core Features
- **Transaction CRUD** - Create, Read, Update, Delete transactions
- **Intelligent Caching** - 5-minute cache with automatic invalidation
- **Multi-tab Sync** - Real-time synchronization across browser tabs
- **Pagination & Search** - Efficient data navigation with instant search
- **Master Data Management** - Categories and Wallets management
- **Responsive Design** - Mobile-first, fully responsive UI
